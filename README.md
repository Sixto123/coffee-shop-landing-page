# Coffee Shop Landing Page

Landing page de una cafetería desarrollada con HTML y CSS.

![Vista del proyecto](./image/preview.png)

## Características
- Página principal con imagen de fondo
- Sección de menú
- Sección nosotros
- Galería de imágenes
- Sección de contacto
- Cuenta con scroll y algunas animaciones sencillas

## Tecnologías
- HTML
- CSS
